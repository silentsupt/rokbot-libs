## ทำอะไรได้บ้าง

- 🚜 **เก็บทรัพย์อัตโนมัติ** — ส่งกองทัพไปเก็บ อาหาร / ไม้ / หิน / ทอง (ตั้งจำนวนกองได้)
- 🛡️ **ฝึกทหารอัตโนมัติ** — ฝึกต่อให้เอง เลือกระดับ T1–T5 แยกแต่ละชนิดทหาร
- 🛒 **ซื้อของในตลาดอัตโนมัติ** — ซื้อไอเทมที่ต้องการ + รีเฟรชฟรีให้เอง
- 🤝 **บริจาคเทคโนโลยีสมาพันธ์** — บริจาคจนครบโอกาสอัตโนมัติ
- 🌙 **เก็บรางวัลล็อกอินรายวัน** — ออก/เข้าเกมเก็บรางวัลตอน 00:00 UTC

## ระบบจัดการ

- 🔌 **รองรับหลายเครื่อง** — เพิ่ม/ลบ/เปลี่ยนชื่อ + เชื่อม ADB อัตโนมัติ
- ⚙️ **ตั้งค่าแยกแต่ละเครื่อง** — เลือกทรัพยากร จำนวนกอง และเปิด-ปิดออโต้แต่ละแบบ
- 🟢 **จุดสถานะเครื่อง** — เชื่อมแล้ว / กำลังเชื่อม / เชื่อมไม่สำเร็จ (ข้ามเครื่องที่ต่อไม่ได้)
- 🔄 **อัปเดตอัตโนมัติ** — มีเวอร์ชันใหม่ดาวน์โหลดให้เอง
