# ROK MULTI-BOT

บอท Rise of Kingdoms ที่ทำขึ้นมาเพื่อให้เล่นแบบไม่ต้องเฝ้า เปิดทิ้งไว้แล้วมันจัดการให้เอง
จะดูแลกี่เครื่องพร้อมกันก็ได้ ตั้งค่าครั้งเดียวแล้วปล่อยมันทำงานยาวๆ

## มันช่วยอะไรคุณได้

เรื่องน่าเบื่อที่ต้องทำซ้ำๆ ทุกวัน ปล่อยให้บอทจัดการแทน:

- **เก็บทรัพย์** — ส่งกองออกไปเก็บอาหาร/ไม้/หิน/ทองให้เรื่อยๆ ไม่ต้องคอยกดเอง
- **ฝึกทหาร** — ช่องว่างเมื่อไหร่ฝึกต่อให้ทันที เลือกได้ว่าจะฝึก T อะไร ทหารแบบไหน
- **ซื้อของในตลาด** — เจอของที่อยากได้กดซื้อให้ แถมรีเฟรชฟรีให้ด้วย
- **บริจาคสมาพันธ์** — บริจาคเทคโนโลยีให้ครบทุกวันแบบไม่ลืม
- **เก็บล็อกอินรายวัน** — ตื่นมาก็ได้รางวัลครบแล้ว (เก็บให้ตอนเปลี่ยนวัน)

## เปิดทิ้งไว้ได้สบายใจ

- หลายเครื่องพร้อมกัน ตั้งค่าแยกใครแยกมันได้ ใครจะเก็บอะไรเปิดออโต้แบบไหนก็เลือกเอง
- เน็ตหลุด เด้งหน้าแปลกๆ หรือเครื่องไหนมีปัญหา มันข้ามแล้วไปต่อเอง ไม่ค้างทั้งระบบ
- มีอัปเดตใหม่เดี๋ยวมันบอกเอง กดอัปทีเดียวจบ
